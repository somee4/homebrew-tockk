# homebrew-tockk

Homebrew tap for [Tockk](https://github.com/somee4/tockk) — notch-style notifications for local AI coding agent events.

## Install

```bash
brew install --cask somee4/tockk/tockk
```

Or tap first, then install:

```bash
brew tap somee4/tockk
brew install --cask tockk
```

## Requirements

- macOS 13 Ventura or later

## Upgrade

```bash
brew upgrade --cask tockk
```

## Uninstall

```bash
brew uninstall --cask tockk
brew untap somee4/tockk
```
